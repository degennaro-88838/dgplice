📊 Sincronizzazione Google Sheets

Per sincronizzare i dati tra tablet e smartphone, collega un Google Sheet. I dati saranno salvati nel foglio e accessibili da qualsiasi dispositivo.

1 Vai su sheets.google.com e crea un nuovo foglio. Rinominalo "AutoNoleggio".
2 Vai su Estensioni → Apps Script, incolla lo script riportato sotto e fai clic su Distribuisci → Nuova distribuzione → App Web. Imposta accesso: Chiunque. Copia l'URL.
3 Incolla l'URL qui sotto e clicca Connetti.
📋 Mostra script Apps Script da incollare 
function doPost(e) {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var data = JSON.parse(e.postData.contents);
  var sheet = ss.getSheetByName(data.foglio) || ss.insertSheet(data.foglio);

  if (data.azione === 'scrivi') {
    sheet.clearContents();
    if (data.righe && data.righe.length > 0) {
      sheet.getRange(1, 1, data.righe.length, data.righe[0].length).setValues(data.righe);
    }
  }
  return ContentService.createTextOutput(JSON.stringify({ok: true})).setMimeType(ContentService.MimeType.JSON);
}

function doGet(e) {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var foglio = e.parameter.foglio;
  var sheet = ss.getSheetByName(foglio);
  if (!sheet) return ContentService.createTextOutput(JSON.stringify({righe:[]})).setMimeType(ContentService.MimeType.JSON);
  var dati = sheet.getDataRange().getValues();
  return ContentService.createTextOutput(JSON.stringify({righe: dati})).setMimeType(ContentService.MimeType.JSON);
}
⚠️ Zona pericolosa

Cancella tutti i dati salvati nel browser. Questa operazione non è reversibile.
